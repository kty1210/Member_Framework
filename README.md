<!-- ABOUT THE PROJECT -->
## 소개
<br/>
<br/>



CRUD를 이용한 회원관리 게시판
<br/>

<br/>
<br/>




구현 메소드
<br/>

* save : 회원가입
* login : 로그인 
* findAll : 전체 회원 조회
* findById : 특정 회원 조
* delete : 회원 삭제
* update : 회원정보 수정


<br/>
<br/>

추가된 라이브러리
<br/>

* lombok
* H2
* JUnit
* MyBatis
* HikariCP
* log4j2
